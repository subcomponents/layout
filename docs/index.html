CSS layout

Abstract

Render one or two column layout according to markup. No classes. Default one column.

Installation

CDN link:

<link rel="stylesheet" href="https://unpkg.com/hyper-layout-css@1.4.2/dist/layout.min.css"/>

NPM package:

$ npm install --save-dev hyper-layout-css

CSS import:

@import url("https://unpkg.com/hyper-layout-css@1.4.2/dist/layout.min.css");
